Game Projects

NEON City

Gameplay Developer

It is a 2D online multiplayer platform and lateral runner videogame set in a cyberpunk city. 

A player has to be the last one standing on a frantic race with a procedural generated level.

The objective of this project was to apply the correct design and optimal development practices learned during the bootcamp based on agile development methodologies.

Skyway Savior

UI Developer | Gameplay Developer | Sound

It is a 3D single player race videogame set in a futuristic circuit. 

In this project I was in charge of implementing the UI for the main and pause menu, as well as selecting the sound effects and background music.

Don't Stop Kiddo!

Game Design | Level Developer | Gameplay Developer

It is a 3D single player infinite runner.

Made on a 9h game jam activity for bootcamp.
I proposed the plot and gameloop design developed with the team. Built the scenario and obstacle generation by game object spawning.
This game design is totally scalable.

Gyro Blitz

Designer and Developer

It is a personal 3D multiplayer sportlike game.

Based on the rules of table hockey, this project aims to take place in a futuristic environment. The flagship goal is to apply a non-symmetric multiplayer and spin mechanic that takes advantage of the accelerometer features of mobile devices.
